# Nicolas Ostermann Portfolio

Modern multilingual developer portfolio built with Next.js, TypeScript and Tailwind CSS.

## Live Website

https://portfolio-weld-one-68.vercel.app

## Features

- Multilingual support (EN, DE, ES, IT, TR)
- Automatic browser language detection
- Manual language switcher
- Dark / Light mode
- Responsive design
- Animated UI with Framer Motion
- Accessibility optimized
- SEO optimized
- Contact form with Resend
- GitHub Actions CI
- Dependabot integration
- ESLint + Prettier + Husky
- Jest snapshot testing
- Lighthouse 100/100/100/100
